Class LogManager

Log Class for Reborn

This class is really adapter only between Monolog and Reborn CMS. Supported logging methods are -
  • debug
  • info
  • notice
  • warning
  • critical
  • alert
  • emergency
You can see defination for above methods at monolog documentation.
