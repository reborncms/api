# get( string $key = '*', mixed $default = null )

Get request param from $_GET (or) $_POST. $_GET (or) $_POST is base on request method. If you want to get all Input Field, use '*' key. example:

// Return all input field
Input::get('*');
// Return input field by key username
Input::get('username');
// Return default value when the key name is null
Input::get('age', 26);
